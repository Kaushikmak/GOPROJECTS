# GOPROJECTS

A centralized repository containing a collection of projects, utilities, and experiments that I implemented in the **Go (Golang)** programming language.

## Repository Overview

This repository operates as a **monorepo**. Each subdirectory contains a distinct, self-contained project with its own logic, dependencies, and documentation.

### Projects Index

| Project Name | Type | Description | Status |
| :--- | :--- | :--- | :--- |
| **[TaskTracker](./TaskTracker)** | CLI Tool | A lightweight, persistent command-line task manager supporting CRUD operations and JSON storage. | Active |
| **[Goshrt](./UrlShortner)** | Web Service | A high-performance URL shortener built with **Go Fiber** & **Redis**, featuring a Neobrutalist frontend and rate limiting. | Active |
